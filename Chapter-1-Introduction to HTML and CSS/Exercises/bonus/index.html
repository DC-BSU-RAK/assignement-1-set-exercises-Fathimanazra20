HTML & CSS Quick Reference Guide

Common HTML Tags

<p> - Paragraph

Used to define paragraphs of text.

<p>This is a paragraph.</p>

<h1> to <h6> - Headings

Used to define headings. <h1> is the largest, <h6> is the smallest.

<h1>Main Title</h1>
<h2>Subheading</h2>

<a> - Anchor

Defines hyperlinks to other pages or locations.

<a href="https://example.com">Visit Site</a>

<img> - Image

Embeds an image in the document.

<img src="image.jpg" alt="Description">

Common CSS Properties

color

Sets the text color.

p {
  color: blue;
}

background-color

Sets the background color of an element.

body {
  background-color: #f9f9f9;
}

font-family

Specifies the font style for the text.

body {
  font-family: Arial, sans-serif;
}
